Enunciado

Em um jogo de palavras cruzadas, o jogador preenche um retângulo com quadrados em branco e pretos, onde cada linha e coluna em branco deve ser preenchida com uma palavra. As palavras horizontais cruzam com as palavras verticais em letras comuns. Dadas uma palavra horizontal e uma palavra vertical, sua tarefa é encontrar a melhor letra de cruzamento entre elas, definida como a letra que permite o cruzamento mais à direita na palavra horizontal e, em caso de empate, o cruzamento mais abaixo na palavra vertical.

Entrada Duas strings em uma lista: a primeira string é a palavra horizontal, e a segunda string é a palavra vertical.

Saída: Uma lista de dois inteiros descrevendo a melhor letra de cruzamento: o índice na palavra horizontal e o índice na palavra vertical. Se não houver cruzamento possível, ambos os inteiros devem ser -1.
